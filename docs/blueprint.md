# **App Name**: Synthfolio

## Core Features:

- Dynamic Content Sections: Display structured sections for 'About Me', 'Education', 'Certifications', 'Work History', and 'Skills' based on provided content, allowing for easy updates and organized presentation.
- Interactive Project Showcase: Present detailed project descriptions including purpose, technologies, skills demonstrated, and links to GitHub or live demos in an engaging card-based format.
- AI-Powered Skill Aligner: Integrate an AI tool where visitors can paste a job description, and the AI will analyze the portfolio content to highlight relevant skills and projects, showcasing AI integration capability.
- Professional Contact Hub: A dedicated section to clearly display professional contact information (email, LinkedIn, GitHub) and encourage connection and collaboration with potential employers.
- Responsive Card-Grid Layout: Implement a modern, card-based, and grid-driven layout that ensures optimal viewing and navigation across various devices, consistent with a single-page scrolling experience and minimal clutter.

## Style Guidelines:

- Scheme: Dark theme, chosen to achieve a sleek, high-contrast, futuristic, and technical interface as specified.
- Primary color: Electric Blue (#007CFF), selected for its vibrant and bold professional appearance, providing excellent contrast and a modern feel against the dark background.
- Background color: Deep Slate (#161C1E), a desaturated, very dark blue (HSL(211, 15%, 10%)) that provides a sophisticated base and enhances the high-contrast aesthetic requested by the user.
- Accent color: Crimson Red (#D7263D), utilized sparingly for interactive elements, crucial highlights, and calls to action to add visual interest and impact.
- Headline font: 'Space Grotesk' (sans-serif), a modern and tech-inspired font that embodies the futuristic and technical branding. Body font: 'Inter' (sans-serif), selected for its excellent readability and neutral, objective appearance, ideal for longer text passages.
- Code snippets font: 'Source Code Pro' (monospace, sans-serif), for clear and structured presentation of any code examples or technical mentions, ensuring high legibility for programming content.
- Utilize minimalist and crisp line icons, with a subtle technical or abstract aesthetic, to maintain the clean and clutter-free interface. SVG formats for sharp, scalable visuals.
- Implement a card-based and grid-based single-page scroll layout, organizing content into distinct, easily digestible blocks for a clean and professional look, mimicking a modern SaaS landing page with minimal visual distractions.
- Subtle and purposeful animations, such as smooth scrolling between sections, gentle fade-in effects for content as it enters the viewport, and refined hover states for interactive elements, enhancing the modern feel without distracting from the content.
